# Do you really need to code before you start with AWS?

As a beginner, you don't need previous programming knowledge to work with AWS services. A basic understanding of IT is a good starting point. That said, you should learn one programming language as you climb the ladder.

It makes things easier, and you can implement more complex solutions once you know a language. But to start, you can work with many AWS services without any programming knowledge. A good place to deepen your understanding is the AWS Well-Architected Framework and the AWS White Papers.

The starting requirement was never a language, it's the willingness to keep leveling up as the problems get harder.
